# AI数字人视频生成系统

本地数字人口播视频生成工具，基于素材模板合成模式，无需网络，无需GPU，快速生成竖屏口播视频。

## 功能

- **本地数字人口播** — 导入真人出镜视频素材，搭配文案自动生成口播视频
- **模板系统** — 保存常用配置（素材、音量、字幕、布局），一键复用
- **自动字幕** — 语音转字幕，自动对齐，无需手动添加
- **TTS配音** — 支持多种音色，一键将文案转为配音
- **商品叠加** — 可在视频角落叠加商品图或品牌元素
- **背景视频** — 支持自定义背景，支持图片或视频背景
- **BGM背景音乐** — 添加背景音乐，支持音量调节
- **一键生成竖屏视频** — 输出 1080×1920 竖屏格式，直接用于抖音/快手/视频号

## 快速开始

### 安装方式

1. 下载 `AI Digital Human Setup.exe` 安装包
2. 运行安装程序，一路下一步即可
3. 启动软件，首次会自动检测 FFmpeg

或直接运行免安装版：

1. 下载 `AI Digital Human.exe`（位于 `win-unpacked` 目录）
2. 双击直接运行，无需安装

### 前置要求

- FFmpeg（软件会自动检测，如未检测到请手动安装）
- Windows 10 及以上系统

## 推荐配置

**最低配置：**
- Windows 10
- 8GB RAM
- FFmpeg（官网下载或 `winget install ffmpeg`）

**推荐配置：**
- Windows 11
- 16GB RAM
- SSD 存储

## 已知问题

- ❌ **不支持换脸** — 当前版本不含换脸功能，纯素材合成
- ❌ **不支持实时直播** — 离线生成模式，非直播推流
- ❌ **不支持 GPU AI 生成** — 不含 AI 图像/视频生成，需自行准备素材
- ✅ **素材模板合成模式** — 核心是素材拼接 + 语音合成，适合有现成素材的用户

## 技术栈

- 前端：Electron + Vue 3 + TypeScript + Vite
- 后端：Python FastAPI + SQLite
- 视频处理：FFmpeg
- 语音合成：Edge TTS
- 视频渲染：FFmpeg 滤镜图合成
